# Corolário

Consequência directa de um [[teorema]]. A prova de um corolário é tipicamente muito curta porque o trabalho pesado já foi feito pelo teorema que o precede.

A palavra vem do latim *corollarium*, "pequena coroa" — uma grinalda oferecida como brinde. O corolário é o brinde que vem com o teorema: não exige esforço substancial, mas é um resultado genuíno.

## Estrutura lógica

Se *A* implica *B* (teorema), e *B* implica *C* por um passo trivial, então *C* é um corolário de *A*.
