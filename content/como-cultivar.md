# Como cultivar este jardim

Diretrizes pessoais para manter este jardim saudavel.

## Tipos de notas

- **Seedling** — Ideia inicial, ainda crua. Poucas frases, muita curiosidade.
- **Budding** — Em crescimento. Tem alguma estrutura, referencias, ligacoes.
- **Evergreen** — Madura, revista, com profundidade. Pode continuar a crescer.

## Convencoes

- **Lingua:** Portugues (PT-PT), salvo citacoes ou termos tecnicos
- **Formato:** Markdown com frontmatter YAML
- **Ligacoes:** Usar `[[wikilinks]]` do Obsidian
- **Datas:** Incluir `date` no frontmatter de cada nota
- **Tags:** Usar `tags` no frontmatter, nao `#tags` no corpo

## Rotinas

1. **Captura** — Qualquer ideia, link ou citacao vai para uma nota seedling
2. **Conexao** — Ao rever notas, perguntar: "a que outras notas se liga isto?"
3. **Cultivo** — Notas seedling que crescem passam a budding
4. **Poda** — Notas obsoletas sao atualizadas ou arquivadas, nao apagadas
