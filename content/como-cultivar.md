# 🌱 Como cultivar este jardim

Algumas diretrizes pessoais para manter este jardim saudável.

## Tipos de notas

- **🌱 Seedling** — Ideia inicial, ainda crua. Poucas frases, muita curiosidade.
- **🪴 Budding** — Em crescimento. Tem alguma estrutura, referências, ligações.
- **🌳 Evergreen** — Madura, revista, com profundidade. Pode continuar a crescer.

## Convenções

- **Língua:** Português (PT-PT), salvo citações ou termos técnicos
- **Formato:** Markdown com frontmatter YAML
- **Ligações:** Usar `[[wikilinks]]` do Obsidian
- **Datas:** Incluir `date` no frontmatter de cada nota
- **Tags:** Usar `tags` no frontmatter, não `#tags` no corpo

## Rotinas

1. **Captura** — Qualquer ideia, link ou citação vai para uma nota *seedling*
2. **Conexão** — Quando revejo notas, pergunto: "a que outras notas se liga isto?"
3. **Cultivo** — Notas *seedling* que crescem passam a *budding*
4. **Poda** — Notas obsoletas são atualizadas ou arquivadas, não apagadas
