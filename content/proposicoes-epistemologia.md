# Proposições — axiomas, teoremas e outras afirmações

A confusão entre estes termos é comum porque o uso quotidiano os trata como sinónimos. Em epistemologia e lógica formal, cada um tem um papel distinto e uma relação específica com a ideia de prova.

A distinção central: **axiomas assumem-se, teoremas provam-se, conjecturas aguardam prova, hipóteses testam-se.**

## Mapa de conceitos

| Conceito | Natureza | Nota |
|---|---|---|
| [[axioma]] | Aceite, não provado | Ponto de partida de um sistema formal |
| [[teorema]] | Provado a partir de axiomas | A prova é o que o distingue |
| [[postulado]] | Aceite, historicamente "pedido" | Hoje equivalente a axioma |
| [[lema]] | Provado, auxiliar | Serve para demonstrar outro resultado |
| [[corolario]] | Provado, consequência directa | O trabalho pesado já foi feito |
| [[conjectura]] | Não provada, suspeita-se verdadeira | Um teorema em espera |
| [[hipotese]] | Assumida para teste ou argumento | Científica: falseável; matemática: premissa local |
| [[tese]] | Defendida, não provada | Domínio da argumentação, não da lógica formal |

## Hierarquia de certeza

Do mais certo ao mais incerto:

1. **[[axioma|Axioma]]** — aceite, não provado
2. **[[teorema]]** — provado a partir de axiomas
3. **[[corolario]]** — provado a partir de um teorema
4. **[[lema]]** — provado, auxiliar de um teorema
5. **[[conjectura]]** — não provada, suspeita-se verdadeira
6. **[[hipotese]]** — assumida para teste ou argumento
