# Lema

[[teorema|Teorema]] auxiliar. Prova-se formalmente, mas o seu interesse é instrumental: serve para demonstrar outro resultado mais importante.

Um lema raramente é citado por si próprio; é uma peça de encaixe numa prova maior. A palavra vem do grego *lêmma*, "algo que se recebe ou se toma" — toma-se o lema como degrau para chegar ao teorema principal.

## Exemplo

O **Lema de Zorn** é um resultado importante da teoria dos conjuntos, mas o seu nome como "lema" reflecte o papel original que desempenhou: Zorn provou-o para usar na demonstração de outro resultado.
